PHP反序列化Payload构造工具

快速构造、修正和编码PHP序列化Payload

本地离线工具 · 自动计算字符串字节长度

构造区

raw array fragment 填连续的“键+值”已序列化片段；raw object fragment 可填 类名|属性名片段+属性值片段，不写类名时默认 User。

常用基础值

键支持 string/int；值支持 string/int/bool/null/raw。

数组结构快捷填充

选择private或protected时，工具会自动按PHP序列化规则处理属性名；界面里只需要填写普通属性名。

对象结构快捷填充

修正功能会扫描 s:数字:"内容"; 和 O:数字:"类名": 的长度，按 UTF-8 字节数重新计算。

先点「解析为对象」会把序列化串反向解析成对象式伪代码；随后可用「对象→拆分类」展开成多个 PHP class，或用「对象→类」生成可继续一键序列化的嵌套类写法。

支持解析 class 中的 public/protected/private 属性及其默认值，自动生成反序列化 Payload。

一键会先按你粘贴的 PHP 类代码自动实例化成内部对象结构，再把这个实例做 serialize，适合从类定义快速得到 Payload。

属性数量篡改

可用于 CVE-2016-7124 一键绕过 PHP < 5.6.25 / < 7.0.10

扫描当前 Payload 中所有 O:N:"类名":，显示每个对象的属性计数并可单独修改。将属性计数改为大于实际属性数即可跳过 __wakeup()。

str_replace / preg_replace 替换逃逸

输入源串和替换规则，自动计算替换后完整串、字节差值。

引用链 (R:reference)

                    R:N = 第N个被反序列化的元素
                  

                  
PHP 按反序列化顺序给对象、数组和字符串值编号。R:1 通常是根对象/根数组；键名和属性名不作为引用目标。替换字段只能指向它之前已经创建的目标，所以第一个属性值通常从
                    R:2 起可被后续字段引用。
